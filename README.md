# Coffee Machine (Python)

A console-based coffee machine simulation built using Python.

## Features
- Order espresso, latte, or cappuccino
- Resource checking before payment
- Coin-based payment system
- Machine commands (info / stop)
- Loop-based system (machine keeps running)

## What I practiced
- Functions and loops
- Dictionaries
- Program flow & state
- Debugging logic errors

Built as part of a Python learning course (Day 15).
